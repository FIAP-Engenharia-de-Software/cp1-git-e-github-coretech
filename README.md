# 🚗 SocialConnect

## 📌 Sobre o Projeto
O **SocialConnect** é um projeto desenvolvido como parte do 1º Checkpoint da disciplina de **Web Development da FIAP**.  

A proposta do projeto melhorar e arrumar o sistema da empersa SocialConnect.

---

## 🏢 Sobre a Empresa
A **Social Connect** é uma empresa inovadora de carros focada no setor automotivo que busca fabricar carros elétricos de alta performance.  

Nosso site oferece carros com a cara de cada cliente.
